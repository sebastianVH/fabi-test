¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo que utiliza el sistema de control de versiones Git. Es el hogar de millones de proyectos de código abierto y privado.

Características principales:

  • Control de versiones: Rastrea cambios en tu código a lo largo del tiempo
  • Colaboración: Permite que múltiples desarrolladores trabajen en el mismo proyecto
  • Repositorios: Espacios donde se almacena y organiza el código
  • Issues: Sistema para reportar problemas y sugerir mejoras
  • Pull Requests: Propone cambios y mejoras al código

¿Por qué usar GitHub?

GitHub te permite:

  • Guardar tu código de forma segura en la nube
  • Colaborar con otros desarrolladores
  • Mostrar tu trabajo y crear un portafolio
  • Contribuir a proyectos de código abierto
  • Aprender de otros desarrolladores
